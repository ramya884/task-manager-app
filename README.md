## ✨ Features

- User Registration
- User Login Authentication
- JWT Token Authentication
- Protected Routes
- Create Tasks
- Delete Tasks
- Update Task Status
- Responsive Dashboard UI
- MongoDB Database Integration

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ramya884/task-manager-app.git
```

---

## Backend Setup

```bash
cd backend
npm install
npm run server
```

Create `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=mysecretkey
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
